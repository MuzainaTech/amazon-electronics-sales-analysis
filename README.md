# 📊 Amazon Electronics Sales Analysis

## Overview

This project analyzes customer reviews from Amazon Electronics products using Python. The objective is to clean the data, perform exploratory data analysis (EDA), and generate meaningful business insights through data visualization.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

---

## Project Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights

---

## Key Visualizations

- Distribution of Customer Ratings
- Top 10 Product Categories
- Top 10 Brands
- Customer Reviews Over Time
- Average Rating by Product Category

---

## Key Insights

- Most customer ratings are positive.
- Some product categories receive significantly more reviews than others.
- Customer engagement varies across brands.
- Review activity changes over time.
- Certain product categories consistently achieve higher average ratings.
